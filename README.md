# 💰 Finance

A cross-platform Flutter application for personal finance management, helping users track expenses, manage budgets, and achieve their financial goals with ease.

---

## 📱 About the Project

**Finance** is a comprehensive mobile and web application built with Flutter/Dart that empowers users to take control of their financial life. The app provides intuitive tools for expense tracking, budget planning, and financial insights, leveraging Flutter's cross-platform capabilities to deliver a seamless experience across Android, iOS, Web, and Desktop platforms.

---

## ✨ Features

- 💸 **Expense Tracking** - Record and categorize your daily transactions
- 📊 **Budget Management** - Set budgets and monitor spending by category
- 📈 **Financial Analytics** - Visualize spending patterns with charts and reports
- 💳 **Multi-Currency Support** - Handle transactions in different currencies
- 🔒 **Secure & Private** - Your financial data stays on your device
- 🌐 **Cross-Platform** - Works on Android, iOS, Web, Windows, Linux, and macOS

## 🚀 Getting Started

### Prerequisites

- [Flutter SDK](https://flutter.dev/docs/get-started/install) (latest stable version)
- [Dart SDK](https://dart.dev/get-dart) (included with Flutter)
- Android Studio / VS Code with Flutter extension
- An Android or iOS device / emulator

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/HARISH4415/Finance.git
   cd Finance
   ```

2. **Install dependencies**
   ```bash
   flutter pub get
   ```

3. **Run the app**
   ```bash
   flutter run
   ```
   
   To run on a specific platform:
   ```bash
   flutter run -d android     # Android
   flutter run -d ios         # iOS
   flutter run -d chrome      # Web
   flutter run -d windows     # Windows
   flutter run -d linux       # Linux
   flutter run -d macos       # macOS
   ```
